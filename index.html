Tournament Manager

Round-robin fixture generator & league table
How it works
  1. Create — Enter a name, choose a format and player count, fill in names (or upload a CSV). Then click Generate.
  2. Share — You'll get a bookmark-friendly link for viewers and a private Edit Code for whoever enters scores.
  3. Enter scores — Use the Fixtures tab. The league table updates live. You can also download the scores as CSV, fill them in offline, and re-upload.
  4. Knockout — For Round Robin, go to League Table once done and click Start Knockout Round to seed into a bracket. For Knockout-only tournaments, the bracket is ready straight away.
  5. Overview — Shows the league table and bracket on one page, ideal for a display screen. Viewers land here by default.
